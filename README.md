Ahoj, tohle je můj MC monitor. Vytvořil jsem jednoduchou aplikaci v Pythonu pomocí knihovny webview.
Zároveň jsem udělal malý lokální webový server pomocí http.server, abych mohl načítat vlastní HTML stránku (index.html).

Server běží na adrese 127.0.0.1:8000 v samostatném vlákně, takže neblokuje běh aplikace.
Potom otevřu okno pomocí webview, které zobrazí stránku z toho lokálního serveru.

V podstatě to funguje jako jednoduchá desktopová aplikace, která zobrazuje moje webové rozhraní.
